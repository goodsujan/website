# Personal Portfolio Website

A production-ready portfolio website for an Electronics & Communication Engineering student.

## features
- **Full Stack**: utilizing Node.js, Express, and SQLite.
- **Responsive**: Mobile-first design.
- **Dark Mode**: Integrated toggle with persistent preference.
- **Contact Form**: Functional backend storage.

## Pre-requisites
- [Node.js](https://nodejs.org/) (Version 14+ recommended)

## Setup Instructions

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run the Server**
   ```bash
   npm start
   ```

3. **Visit Site**
   Go to [http://localhost:3000](http://localhost:3000)

## Project Structure
- `public/`: Contains all frontend code (HTML, CSS, JS).
- `server.js`: The backend server file.
- `database.js`: Database configuration.

## Customization
- Edit `public/index.html` to update your personal details.
- Place your resume at `public/assets/resume.pdf`.
